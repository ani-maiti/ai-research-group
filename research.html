Our Teams Research Focus

We explore topics in Artificial Intelligence, Deep Learning, NLP, Multi-Modal Learning, and AI for Scientific and Medical Applications. Our goal is to develop innovative solutions using knowledge distillation, structured representations, fusion, agentic AI systems, and low-resource AI.

Ongoing Research

AI-Driven Agents for Sequential Code Execution

We are creating database for and analyzing - AI-generated code execution through pre-trained and API-based large models.

Key Focus:

  • ✔ We are creating a structured dataset of AI-generated code executions.
  • ✔ We are building a framework for future studies on AI-driven programming refinement.

Knowledge Distillation - Exploring Methods for Calculating Loss

Student: Samuel Adewumi (IBM Masters Fellow)

This project investigates various loss functions (token-level, feature-level, and patch-level) for improving the transfer of knowledge from large Vision Transformers (ViT) to smaller student models using PyTorch.

Key Focus:

  • ✔ Benchmarking different loss functions for knowledge distillation.
  • ✔ Evaluating performance on small ViT like models.

Evaluating Structured Intermediate Representations (JSON) in NLP

Students: Temesgen Alema Tikure & Zichun Wang

This study explores whether introducing structured intermediate representations (JSON) improves semantic preservation when converting between text and embeddings.

Key Focus:

  • ✔ Comparing text embeddings with JSON-structured embeddings.
  • ✔ Using language models for reconstruction and analysis.

Minimal Fine-Tuning for Multi-Modal Deep Learning

Student: Renata Castellanos

Supported by: NASA Space Grant Research Enhancement Award

This project focuses on developing a low-resource AI pipeline for fine-tuning deep learning models on point cloud and image data, making research accessible to students with limited computing power.

Key Focus:

  • ✔ Using Waymo Open Dataset & KITTI for point cloud fusion.
  • ✔ Fine-tuning pretrained models with minimal resources.
  • ✔ Investigating low-resource training techniques.

3D Object Detection

Applying a deep learning-based framework that improves long-range 3D object detection even in challenging environments.
